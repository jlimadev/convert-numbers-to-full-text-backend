# Spell Out Numbers

This is a Typescript/NodeJs api where you can call the route with a number and receive it as full text.

## Install the dependencies

```bash
yarn
```

## Start the app

```bash
yarn start
```

## Testing

```bash
yarn test
```

Route: `/number/:number`

```
curl http://localhost:3000/numbers/888
curl http://localhost:3000/numbers/10
```
